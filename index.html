Our Profile

L'StudyGroups is a collaborative learning community formed as part of the Coding Camp program. This group provides a supportive environment for learning web development through structured materials, hands-on practice, and teamwork.

We focus on mastering HTML, CSS, and JavaScript while building real projects that reflect real-world website development.
